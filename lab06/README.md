## Математическая часть

### 1) Дискретная случайная величина
Пусть задан ряд распределения:

`P(X = x_i) = p_i, i = 1..k, sum(p_i)=1`

Вычисляются:

- Эмпирическая вероятность:

`p_i* = n_i / N`

- Выборочное среднее:

`m* = (1/N) * sum(x_j)`

- Выборочная дисперсия (с делением на `N`):

`D* = (1/N) * sum((x_j - m*)^2)`

- Относительная погрешность:

`delta = |estimate - reference| / |reference| * 100%`

- Статистика хи-квадрат:

`chi2 = sum((n_i - N*p_i)^2 / (N*p_i))`

Критерий: гипотеза о согласии принимается, если `chi2 < chi2_critical(df=k-1, alpha=0.05)`.

### 2) Нормальная случайная величина
Реализована генерация методом Бокса-Мюллера:

- `U1, U2 ~ Uniform(0,1)`
- `Z1 = sqrt(-2 ln U1) * cos(2*pi*U2)`
- `X = m + sigma * Z1`, где `sigma = sqrt(D)`

Вычисляются:

- выборочные `m*`, `D*`;
- относительные погрешности относительно заданных `m`, `D`;
- критерий хи-квадрат (бины формируются по равновероятностным квантилям теоретического нормального распределения).

## Установка и запуск

1. Создать и активировать виртуальное окружение:

```powershell

.\.venv\Scripts\Activate.ps1
```

```powershell

```
